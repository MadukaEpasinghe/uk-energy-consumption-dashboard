# ⚡ UK Energy Consumption Dashboard

> 📊 An interactive dashboard analyzing energy consumption trends to support data-driven decision-making

---

## 📌 Overview
This project presents a Power BI dashboard that explores energy consumption patterns across the UK. It provides insights into usage trends, seasonal variations, and potential inefficiencies.

---

## 🧠 Business Problem
Energy providers and policymakers need clear insights into consumption patterns to:
- Optimize energy distribution
- Reduce costs
- Improve sustainability

---

## 💡 Solution
An interactive dashboard was developed to:
- Monitor energy consumption trends
- Identify peak usage periods
- Analyze seasonal patterns

---

## 📊 Key Insights
- Energy consumption peaks during winter months
- Significant variation across time periods
- Opportunities exist for optimizing energy usage

---

- Power BI
- Data Cleaning & Transformation
